  # Installation
    npm install bcrypt connect-session-sequelize dotenv express express-handlebars express-session mysql2 sequelize
  # Usage
    You will need to create a .env file and add 
    DB_NAME='picturfolio_db' 
    DB_USER='root'
    DB_PW='(your password)'
